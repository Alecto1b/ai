Peringatan: Jalankan Aplikasi Menggunakan Python Server

Anda membuka file HTML secara langsung (file://). Browser memblokir request API langsung dari file lokal karena kebijakan keamanan CORS (Cross-Origin Resource Sharing).

Opsi A: Melalui VS Code (Sangat Direkomendasikan)

  1. Buka proyek di editor VS Code Anda.
  2. Tekan tombol shortcut Ctrl + Shift + D untuk membuka panel Run & Debug.
  3. Pilih menu dropdown: Full Stack: Server & Firefox (atau Chrome).
  4. Tekan tombol F5 di keyboard Anda.

Opsi B: Melalui Terminal

  1. Buka terminal Anda di direktori proyek ini.
  2. Jalankan server menggunakan perintah: python server.py
  3. Buka browser dan buka alamat: http://localhost:8000
